# Utils

## Time Utility
To compare time scalability of **naive** approach vs **kapra** approach, launch the *test* utility:
```console
foo@bar:~$ cd Utility
foo@bar:~$ ./test.sh
```

## Clean Utility
To clean all generated files, output and datasets, launch the *clean* utility:
```console
foo@bar:~$ cd Utility
foo@bar:~$ ./clean.sh
```
